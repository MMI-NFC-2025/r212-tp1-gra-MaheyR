# R212-2026-TP1

Ce dépôt contient le code source et les ressources associées pour le TP1 du cours R212-2026.

## Contenu du dépôt

- `src/` : Contient le code source du projet.
- └─`assets/` : Ressources statiques telles que les images et les polices.
- └─`components/` : Composants réutilisables de l'application.
- └─`layouts/` : Modèles de mise en page pour les pages.
- └─`pages/` : Pages principales de l'application.
- └─`styles/` : Fichiers de styles CSS.
- `public/` : Contient les fichiers publics accessibles.
- `astro.config.mjs` : Fichier de configuration pour Astro.
- `package.json` : Fichier de gestion des dépendances et scripts npm.
- `README.md` : Ce fichier de documentation.

## Instructions pour l'exécution

1. Installer les dépendances :
   ```bash
   npm install
   ```
2. Démarrer le serveur de développement :
   ```bash
   npm run dev
   ```
3. Ouvrir votre navigateur et accéder à `http://localhost:4322` pour voir l'application en cours d'exécution.
